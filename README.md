# 📅 ごみ収集カレンダー (Garbage Collection Calendar)

スマートフォンでの利用に最適化された、シンプルで高機能なごみ収集日管理 Web アプリケーションです。  
**1 枚の HTML ファイルだけで動作**し、インストール不要ですぐに使えます。

---

## ✨ 特徴 (Features)

- **カレンダー表示**  
  月ごとの収集スケジュールを一目で確認できます。

- **明日のお知らせバナー**  
  翌日に収集がある場合、画面上部にバナーで通知します。  
  「明日、何ごみの日だっけ？」をすぐに確認できます。

- **柔軟なルール設定**  
  ごみの種類ごとに以下のようなルールを自由に設定できます：  
  - 「毎週 月・木」
  - 「第1・第3 水曜日」
  - 「第2・第4 金曜日」
  - 「隔週」的なパターンも指定週の組み合わせで表現可能  
  さらに、**分類（燃やすごみ／資源ごみなど）を自分で追加・削除**できます。

- **完全ローカル動作**  
  設定情報はブラウザの LocalStorage に保存されます：
  - サーバー不要
  - ユーザー登録不要
  - データは端末内にのみ保存

- **ダークモード対応**  
  OS / ブラウザの設定に合わせて自動的にダークテーマに切り替わるため、夜間でも目に優しい表示になります。

- **PWA 対応（簡易）**  
  `manifest.json` と `icon.png` を配置すると、  
  iPhone (Safari) や Android (Chrome) の「ホーム画面に追加」から、  
  ネイティブアプリのようにフルスクリーンで利用できます。

---

## 🚀 使い方 (Usage)

1. **ファイルを配置**
   - `index.html` をローカルに保存します。
   - 必要に応じて同じフォルダに `manifest.json` と `icon.png` を置きます。

2. **ブラウザで開く**
   - PC なら `index.html` をダブルクリックするだけで OK です。
   - スマホなら、サーバー等にアップロードして URL として開きます。

3. **ルールを設定**
   - 右上の「⚙（設定）」アイコンをタップ。
   - 「燃やすごみ」「ペットボトル」などの分類ごとに、
     - 毎週／第1・第3 週などのモード
     - 曜日（例：月・木）
     - 対象週（例：第1・第3）  
     を設定します。

4. **カレンダーで確認**
   - カレンダー上の日付に、ごみの種類ラベルが表示されます。
   - 日付をタップすると、その日の収集内容の詳細がモーダルで表示されます。

5. **明日のごみを確認**
   - 明日収集がある場合、画面上部にカラフルなバナーでお知らせが表示されます。

## 📂 ファイル構成 (File Structure)

PWA（アプリのようにインストールして使う形）として利用する場合の、推奨構成は以下の通りです。

```text
.
├── index.html      # アプリ本体 (このファイル)
├── manifest.json   # PWA 設定ファイル（Android 向けなど）
└── icon.png        # アプリアイコン (512x512 推奨)
```

> **Note**  
> `manifest.json` と `icon.png` がなくてもアプリ自体は動作します。  
> ただし、Android で「ホーム画面に追加」した際のアイコン表示や、  
> 起動画面などを整えたい場合は配置をおすすめします。

## 🌐 デプロイ方法 (Deployment)

このアプリは _完全に静的な HTML サイト_ なので、以下のようなサービスで手軽に公開できます。

### GitHub Pages

1. GitHub リポジトリを作成
2. `index.html`（と必要なら `manifest.json`, `icon.png`）をコミット
3. GitHub Pages を有効化
4. 公開された URL をスマホから開くだけ

### Vercel / Netlify など

1. 管理画面から新しいプロジェクトを作成
2. `index.html` をアップロード（または GitHub リポジトリを連携）
3. 自動的にデプロイされ、URL が発行されます

## 📱 スマホへのインストール方法 (Add to Home Screen)

### iPhone (iOS / Safari)

1. Safari でこのアプリの URL を開く
2. 画面下の共有ボタン（□ から ↑ のアイコン）をタップ
3. 「ホーム画面に追加」を選択
4. 名前を確認して「追加」をタップ

### Android (Chrome)

1. Chrome でこのアプリの URL を開く
2. 右上のメニュー（︙）をタップ
3. 「アプリをインストール」または「ホーム画面に追加」を選択
4. 案内に従って追加

## 💾 データ保存について (Data Persistence)

- ごみ分類・収集ルールは、ブラウザの `localStorage` に保存されます。
- ブラウザを閉じても設定は保持されます。
- 端末やブラウザを変えると設定は引き継がれません（サーバー連携は行っていません）。

設定を初期状態に戻したい場合は：

1. アプリ内の「初期化」ボタンを押す  
   または  
2. ブラウザのサイトデータ（キャッシュ／ローカルストレージ）を削除する

## 🔐 プライバシー (Privacy)

- このアプリはサーバー通信を行いません。
- 入力されたデータはすべて **ローカルのブラウザ内にのみ保存** されます。
- アクセス解析やトラッキング等も一切行っていません。
